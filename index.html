Projects

Smart Bin

Concept for a smart trash bin that automates waste sorting.

With the rapid growth of urban waste, traditional manual sorting suffers from low accuracy, high labor costs, and poor public participation. SmartBin addresses these challenges by integrating OpenMV-based vision recognition and servo-driven automatic sorting, achieving over 92% classification accuracy and rapid response. Through intelligent guidance and automation, the system significantly improves recycling efficiency and promotes sustainable waste management.

Measure System

A non-contact automated measurement system for quick and accurate object inspection.

This project developed a non-contact automated measurement system for manufacturing and educational scenarios, capable of quickly measuring object shapes, dimensions, and center-point distances. The system integrates an OpenMV vision module with ultrasonic ranging, supporting multiple geometric shapes and basic colors, and achieves approximately 90% measurement accuracy with efficient and stable operation.

Safeguard

A modular public safety sensing system for real-time health and environmental monitoring.

This project developed a modular public safety sensing system designed for post-pandemic public spaces. Core modules include infrared non-contact temperature measurement, temperature and humidity monitoring, smoke detection, Bluetooth communication, and voice alerts. The system supports flexible module combinations to monitor human health and environmental safety in real time, achieving over 95% detection accuracy and stable operation across different scenarios. The project was granted a national utility model patent.

Signals

A device for precise signal distortion measurement using FFT and THD analysis.

This device features three main modules: a signal generator (1 kHz–100 kHz, 5 mV–1 V), an adjustable-gain amplifier (0–40 dB, 1 MHz–20 MHz), and a data processor that performs FFT to calculate total harmonic distortion (THD). The system enables accurate and efficient characterization of signal distortion.
